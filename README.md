# Bandwidth Homebrew Tap

Official [Homebrew](https://brew.sh) tap for Bandwidth CLI tools.

## Install

```bash
brew install Bandwidth/tap/band
```

## Update

```bash
brew upgrade band
```

## Formulas

| Formula | Description |
|---------|-------------|
| `band`  | Bandwidth CLI — manage voice, messaging, numbers, and more |

## How it works

Formulas in this tap are automatically updated by [GoReleaser](https://goreleaser.com) on each release of [Bandwidth/cli](https://github.com/Bandwidth/cli).
